# 🛡️ DrowsiGuard: Smart Driver Safety System

DrowsiGuard is an advanced, AI-powered **Driver Drowsiness and Distraction Detection** system. It uses computer vision in the browser to monitor driver alertness and provides real-time alerts both visually and through a progressive auditory system (Beeps & Speech).

---

## 🚀 Key Features

-   **Browser-Based AI**: Real-time face landmarker tracking (MediaPipe) running directly in the browser (no data leaves your device).
-   **Multi-Stage Alerts**: Loud square-wave beeps followed by synthesized speech warnings ("Please focus on the road").
-   **Dynamic Dashboard**: Live monitoring of driving sessions, detailed event logs, and safety metrics.
-   **Personalized AI Suggestions**: Dynamic safety recommendations generated by local LLMs (Ollama) or cloud APIs (Cohere).
-   **Full-Stack Architecture**: Modern React frontend with a high-performance FastAPI backend and PostgreSQL database.

## 💻 Local Setup

### 1. Prerequisites
- **Node.js** (v18+)
- **Python** (v3.12+)
- **PostgreSQL** (Installed and running)
- **Ollama** (Optional - for local AI suggestions)

### 2. Backend Setup
1. Navigate to the backend folder: `cd backend`
2. Create and activate a Virtual Environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file from the template:
   ```env
   DATABASE_URL=postgresql://user:password@localhost:5432/driver_db
   JWT_SECRET=your_secret_key
   COHERE_API_KEY=optional_for_cloud_ai
   ```
5. Run the server: `uvicorn app.main:app --reload`

### 3. Frontend Setup
1. Navigate to the frontend folder: `cd frontend`
2. Install dependencies: `npm install`
3. Create a `.env.local` file:
   ```env
   VITE_API_BASE=http://localhost:8000
   ```
4. Start the development server: `npm run dev`
